Graphing

The solver includes a graphing calculator that can graph many functions.

Supported Graphing Modes

  • Normal (variable: x)
  • Polar (variable: theta)
  • Implicit (variables: x, y)

Type an expression in the equation editor and press enter. It will graph the expression on the right.
